Pour se faire, voici les differentes commandes disponibles dans notre jeu

Utiliser une potion

Pour utiliser une potion il faut utiliser la commande drink.
Boire une potion permet au joueur de regagner de la vie selon la potion bue. Les potions sont fixes dans les salles et lorsque le joueur boit une potion, celle-ci est retire de la salle.
Pour utiliser cette commande vous devez taper :
  • "drink potion" si il y a une potion dans la salle
  • "drink soin" si il y a un soin dans la salle

Utiliser un objet

Pour utiliser un objet particulier, comme la EnderPearl, vous devez utiliser la commande utiliser ou la commande charger
  • Pour utiliser la commande charger vous devez simplement taper :
    "charger + nom de l'objet"
    Par exemple :
    "charger EnderPearl"
  • Pour utiliser la commande utiliser vous devez simplement taper :
    "utiliser + nom de l'objet"
    Par exemple :
    "utiliser EnderPearl"

Se teleporter

Pour pouvoir vous teleporter vous devez avant remplir un certain nombre de quete et posseder la EnderPearl.
Si et seulement si vous possedez la EnderPearl et que cette derniere est charge, alors vous pouvez vous teleporter.
Pour utiliser la commande teleporter vous devez tapez :
"teleporter + nomde la salle"
Par exemple :
"teleporter temple"
