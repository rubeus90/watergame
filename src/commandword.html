WATER GAMES

Liste des commandes

Obtenir des informations sur la salle

Pour obtenir des informations sur la salle ( liste des objets, listes des potions, les ennemis etc ) vous pouvez soit utiliser le bouton "regarder" soit utiliser la commandde look. Pour utiliser la commande look vous devez taper :
"look"

Se deplacer

Pour se deplacer dans le jeu, vous pouvez soit : utiliser les boutons sur le cote droit en bas de l'interface ( Ouest - Est - Nord - Sud ) ou bien utiliser la command go.
Pour utiliser la commande go vous devez taper :
"go + direction"
Les diretions sont comme pour les boutons : nord, sud, ouest, est, toujours en minuscule.

Revenir en arriere

Vous pouvez revenir en arriere dans le jeu. Pour cela vous pouvez soit utiliser le bouton "BACK" soit utiliser la commande back.
Pur utiliser la commande back vous devez taper :
"back"

Utiliser un objet

Dans le jeu vous pouvez ramasser un certain nombre d'objet. Le nombre d'objet que vous pouvez porter depend de votre sante. Plus votre sante est elevee plus vous pouvez porter d'objet.
Les objets presents dans une salle sont affiches en haut a gauche du jeu. Pour interagir avec les objets vous avez plusieurs possibilitees.Soit vous utiliser l'interface graphique, soit vous utiliser les commandes utilisateurs.
  • Si vous voulez utiliser l'interface graphique, il suffit de cliquer sur l'objet qui vous interesse. Une fois selectionne, vous devez appuyer sur le bouton "prendre" pour prendre l'objet
    L'objet apparait alors dans votre inventaire en bas.

  • Si vous voulez utiliser les commandes utilisateurs, vous devez utiliser les commandes take et drop. Vous devez alors taper :
    "take + nom de l'objet", pour en prendre un objet.
    "drop + nom de l'obet", pour jeter un objet.
    Les noms des objets presents dans une salle sont disponible en haut a droite du jeu.

Utiliser une potion

Pour utiliser une potion il faut utiliser la commande drink.
Boire une potion permet au joueur de regagner de la vie selon la potion bue. Les potions sont fixes dans les salles et lorsque le joueur boit une potion, celle-ci est retire de la salle.
Pour utiliser cette commande vous devez taper :
  • "drink potion" si il y a une potion dans la salle
  • "drink soin" si il y a un soin dans la salle

Se teleporter

Pour pouvoir vous teleporter vous devez avant remplir un certain nombre de quete et posseder la EnderPearl.
Si et seulement si vous possedez la EnderPearl et que cette derniere est charge, alors vous pouvez vous teleporter.
Pour utiliser la commande teleporter vous devez tapez :
"teleporter + nomde la salle"
Par exemple :
"teleporter temple"

Avoir de l'aide

Si vous avez besoin d'aide, vous pouvez taper la commande help

Quitter le jeu

Si vous voulez quitter le jeu, vous pouvez taper la commande quit
